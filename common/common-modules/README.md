# Common Modules 模块

## 概述

通用工具模块，包含业务无关的通用工具类和组件，提供基础的工具方法、常量定义等。

## 主要功能

- **工具类**: 提供各种通用工具方法
- **常量定义**: 定义系统常用的常量
- **JWT 工具**: 提供 JWT 相关的工具方法
- **JSON 处理**: 基于 FastJSON2 的 JSON 处理工具

## 核心组件

### 常量

- `SecurityPathConstants`: 安全路径常量定义

### 工具类

- 各种业务无关的通用工具类

## 依赖说明

- FastJSON2
- MyBatis Plus
- Spring Security OAuth2 JWT
- Swagger Annotations
- Lombok

## 版本要求

- Java 21+
- Spring Boot 3.x
